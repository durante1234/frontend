비동기(async)=>함수가 promise 리턴 / 대기(await) => 실행을 일시 중지하고 promise가 끝날 때까지 기다림

비동기

- 다른 함수와 병렬로 실행되는 함수를 비동기라고 합니다
- setTimeout()


비동기 프로그래밍을 사용하면 javascript에서 오래 실행되는 작업을 시작하고
다른 작업을 병렬로 계속 실행할 수 있습니다.
단점 : 작성하기 어렵고 디버깅 하기가 힘듭니다
그래서 최신 비동기는 콜백을 사용하지 않습니다
대신 promise를 사용하여 해결합니다

promise

- 코드생성 : 시간이 걸릴 수 있는 코드
- 소비코드 : 결과를 기다려야 하는 코드
- promise는 producing코드와 Consumer코드를 연결하는 객체입니다
- Promise 객체 속성
1) Pending (보류중[한정되지 않은])
2) Fulfilled (결과값)
3) Rejected (거부됨)

.then() 결과값에 대한 콜백과 실패에 대한 콜백이라는 두 개의 인수를 받습니다
